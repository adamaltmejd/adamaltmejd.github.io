## Registry Research Toolkit

Tools for working with Swedish registry microdata on
[SCB MONA](https://www.scb.se/mona): metadata search and mock data generation
for developing code outside the secure environment. Built for use by AI coding
agents.

- **regmeta** — search and query SCB registry metadata
- **mock-data-wizard** — generate mock CSV data from MONA projects without exporting personal data

[View on GitHub](https://github.com/adamaltmejd/registry-research-toolkit)

## adamaltmejd.se

This website. A Hugo static site with a custom theme, automated CI/CD, and a publication pipeline that generates citation data from BibTeX and renders a CV to PDF via pandoc and typst. Built to be maintainable by AI coding agents.
